.. _`r-d3heatmap`:

r-d3heatmap
===========

|downloads|

Create interactive heat maps that are usable from the R console, in the 'RStudio' viewer pane, in 'R Markdown' documents, and in 'Shiny' apps. Hover the mouse pointer over a cell to show details, drag a rectangle to zoom, and click row/column labels to highlight.

======== ===========
Home     https://github.com/rstudio/d3heatmap
Versions 0.6.1.1
License  GPL-3 | file LICENSE
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-d3heatmap
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-d3heatmap

and update with::

   conda update r-d3heatmap



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-d3heatmap.
