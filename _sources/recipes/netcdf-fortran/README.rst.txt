Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install netcdf-fortran

and update with::

   conda update netcdf-fortran



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/netcdf-fortran.
