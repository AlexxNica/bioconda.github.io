.. _`r-venn`:

r-venn
======

|downloads|

Draws and displays Venn diagrams up to 7 sets, and any Boolean union of set intersections.

======== ===========
Home     
Versions 1.2
License  GPL (>= 2)
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-venn
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-venn

and update with::

   conda update r-venn



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-venn.
