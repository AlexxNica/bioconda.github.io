paraview
========

|downloads|

ParaView is an open-source, multi-platform data analysis and visualization application based on Visualization Toolkit (VTK).

======== ===========
Home     http://www.paraview.org
Versions 5.2.0
License  BSD
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/paraview
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install paraview

and update with::

   conda update paraview



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/paraview.
