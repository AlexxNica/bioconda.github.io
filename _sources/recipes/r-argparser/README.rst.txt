.. _`r-argparser`:

r-argparser
===========

|downloads|

Cross-platform command-line argument parser written purely in R with no external dependencies. It is useful with the Rscript front-end and facilitates turning an R script into an executable script.

======== ===========
Home     https://bitbucket.org/djhshih/argparser
Versions 0.4
License  GPL (>= 3)
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-argparser/0.4
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-argparser

and update with::

   conda update r-argparser



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-argparser.
