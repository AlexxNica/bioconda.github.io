r-tsne
======

|downloads|

A "pure R" implementation of the t-SNE algorithm.

======== ===========
Home     https://github.com/jdonaldson/rtsne/
Versions 0.1_3
License  GPL
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-tsne
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-tsne

and update with::

   conda update r-tsne



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-tsne.
