.. _`r-itertools`:

r-itertools
===========

|downloads|

Various tools for creating iterators, many patterned after functions in the Python itertools module, and others patterned after functions in the 'snow' package.

======== ===========
Home     
Versions 0.1_3
License  GPL-2
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-itertools
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-itertools

and update with::

   conda update r-itertools



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-itertools.
