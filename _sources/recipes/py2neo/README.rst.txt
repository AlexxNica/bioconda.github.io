py2neo

Python client library and toolkit for Neo4j

======== ===========

Versions 3.1.2
License  Apache Software License
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/py2neo
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install py2neo

and update with::

   conda update py2neo

A Docker container is available at https://quay.io/repository/biocontainers/py2neo.
