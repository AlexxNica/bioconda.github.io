.. _`r-rgraphics`:

r-rgraphics
===========

|downloads|

Data and Functions from the book R Graphics, Second Edition. There is a function to produce each figure in the book, plus several functions, classes, and methods defined in Chapter 8.  

======== ===========
Home     https://www.stat.auckland.ac.nz/~paul/RG2e/index.html
Versions 2.0_14
License  GPL
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-rgraphics
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-rgraphics

and update with::

   conda update r-rgraphics



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-rgraphics.
