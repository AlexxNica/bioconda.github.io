r-reshape2
==========

|downloads|

Flexibly restructure and aggregate data using just two functions: melt and 'dcast' (or 'acast').

======== ===========
Home     https://github.com/hadley/reshape
Versions 1.4.2
License  MIT + file LICENSE
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-reshape2
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-reshape2

and update with::

   conda update r-reshape2



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-reshape2.
